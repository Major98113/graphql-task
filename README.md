<h1>Grapgh-App</h1>
<p>
     The server support 2 scenarios:
     <br/>Show List of Repositories 
     <br/>Show Repo details
</p>
     
<p>To start app locally:</p>
<code>
     docker-compose up -d
</code>

<p>To stop app locally:</p>
<code>
     docker-compose down
</code>
<br/>
<br/>
<p><strong>App will work on 8080 port</strong></p>
